### 建立虛擬環境（可選）

在您的工作項目中建立一個 python virtualenv。

```bash
python -m venv .venv
```
然後啟動它。

<Tabs items={['Debian/Unix', 'Winodws']} defaultIndex={0}>
  <Tab>
    ```bash
    source .venv/bin/activate
    ```
  </Tab>
   <Tab>
    ```bash
    .venv/bin/activate.bat
    ```
  </Tab>
</Tabs>


### Install guerrilla

<Tabs items={['pip', 'pipx', 'poetry', 'pdm']} defaultIndex={0}>

### 導入`Device`類

```py
from guerrilla import Device
```

### 設定配置

將`protocol`和`type`修改為您的裝置要使用的參數。

```py {3-4}
config = {
    "name": "EDR-G9010",
    "type": "mxros",
    "protocol": "ssh",
    "host": "192.168.127.254",
    "username": "admin",
    "password": "moxa",
}
```
> 有關配置的更多詳細信息，請參閱[配置](/docs/guide/configuration)。

### 連接到裝置

```py
device = Device(**config)
device.connect()
```

您將在控制台輸出中看到日誌記錄資訊。

### 開始使用 `guerrilla`

直接執行“show version”指令或透過內建函數執行。

```py
device.run('show version')
```

```py
device.show_version()
```
