# 開始使用

Guerrilla 是一個網路設備設備 SDK，可以輕鬆的與網路設備交互，以純 Python 和可讀的方式編寫腳本和測試。

Guerrilla 結合了甜美的 python 腳本的開發人員體驗，它利用向您的裝置發送命令來進行 CLI 函數調用，按需獲取返回輸出。

## 安裝

- [前置需求](/docs/quick-start/requirement):
  需要 Python 3.12 或是更新的版本.

- [安裝步驟](/docs/quick-start/installation):
  安裝 Guerrilla 套件使用 pip 或是其他套件管理器.




## 致謝

Guerrilla 的開發之所以能夠發展，完全得益於這些令人驚嘆的項目的靈感和想法。

- [Paramiko](https://github.com/paramiko/paramiko) - 領先的原生 Python SSHv2 協定庫。
- [Netmiko](https://github.com/ktbyers/netmiko) - 用於簡化 Paramiko SSH 連接到網路設備的多供應商庫。
- [PySerial](https://github.com/pyserial/pyserial) - Python是串列埠存取函式庫
