# Device Support

Guerrilla is built with modern Pyhton and uses session to add support for any Devices that can connect via SSH, Telnet and Serial.

## Feature Matrix

Based on the above criteria, the following devies are supported for build-in functions:

Test with EDR-G9010 and TN-4908.

#### System Operation
| CLI Command | Description | Guerrilla SDK Usage  |
|-|-|-|
| reload factory-default | Reloads the router device to factory default settings. | device.reload\_factory\_default()    |
| reload factory-default no cert | Reloads the router device to factory default settings without certificate. | device.reload\_factory\_default\_no\_cert()  |
| reload | Reloads the router device. | device.reload()  |
| copy running-config | Exports the configuration file from the device using `tftp` method. | device.export_config('tftp', '192.168.127.3', 'default.ini')  |
| copy running-config | Exports the configuration file from the device using `scp` method. | device.export_config('scp', '192.168.127.3', 'default.ini', 'admin', 'admin')  |
| copy running-config | Exports the configuration file from the device using `sftp` method. | device.export_config('sftp', '192.168.127.3', 'default.ini', 'admin', 'admin')  |
| copy running-config | Exports the configuration file from the device using `usb` method. | device.export\_config('usb', cfg\_path_name='default.ini')  |
| copy device-firmware | Uploads firmware to the device using `tftp` method. | device.upload_firmware('tftp', '192.168.127.3', 'firmware.rom')  |
| copy device-firmware | Uploads firmware to the device using `scp` method. | device.upload_firmware('scp', '192.168.127.3', 'firmware.rom', 'admin', 'admin')  |
| copy device-firmware | Uploads firmware to the device using `sftp` method. | device.upload_firmware('sftp', '192.168.127.3', 'firmware.rom', 'admin', 'admin')  |
| copy device-firmware | Uploads firmware to the device using `usb` method. | device.upload_firmware('usb', filename='firmware.rom')  |
| copy config-file | Imports the configuration file to the device using `tftp` method. | device.import_config('tftp', '192.168.127.3', 'default.ini')  |
| copy config-file | Imports the configuration file to the device using `scp` method. | device.import_config('scp', '192.168.127.3', 'default.ini', 'admin', 'admin')  |
| copy config-file | Imports the configuration file to the device using `sftp` method. | device.import_config('sftp', '192.168.127.3', 'default.ini', 'admin', 'admin')  |
| copy config-file | Imports the configuration file to the device using `usb` method. | device.import\_config('usb', cfg\_path_name='default.ini')  |
| save | Save the current configuration and return to the main menu. | device.save()  |
