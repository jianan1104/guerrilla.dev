## Method 2: Load Configuration from a YAML File

建立一個名為`config.yaml`的文件，其中包含以下內容：

```yaml showLineNumbers filename="config.yaml"
name: EDR-G9010
type: mxros
protocol: ssh
host: 192.168.127.254
username: admin
password: moxa
```

此 YAML 檔案包含裝置的設定。

### 用於載入 YAML 配置的 Python 程式碼

要讀取 YAML 文件，您將需要“PyYAML”庫。 如果尚未安裝，請使用 pip 安裝：

```bash
pip install PyYAML
```
然後，使用以下 Python 腳本從`config.yaml`載入設定：

```py
from guerrilla.utils import load


config = load('config.yaml')
print(config)
```

此腳本導入「load」模組並定義一個函數 load 以從指定路徑讀取 YAML 檔案。
`guerrilla.utild.load` 函數將檔案內容解析為 Python 字典。 最後，腳本讀取“config.yaml”並列印其內容。

確保 YAML 檔案的路徑與腳本中指定的路徑相符。
此範例假設 YAML 檔案與 Python 腳本位於同一目錄中。
